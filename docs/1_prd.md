| 항목 | 내용 |

## 2. 타겟 고객

### 주요 타겟
- **인바운드 외국인 관광객** (일본인, 동남아, 서양인) — 한국 로컬 문화를 깊게 체험하고 싶은 FIT 여행자
- **국내 여행자** — 뻔한 관광지가 아닌 스토리텔링 있는 코스를 원하는 2030 MZ세대
- **가족 단위 여행객** — 아이와 함께 미션을 수행하며 역사·문화를 학습하고 싶은 부모

### 보조 타겟 (B2B)
- 여행사, 호텔, 숙박업소 (단체 키트 납품)
- 학교, 교육기관 (현장학습 패키지)

---

## 3. 핵심 가치 제안

```
한국 전래동화 캐릭터와 함께,
QR로 시작하는 나만의 현장 미션 여행.
도장 깨기 → LP(전설포인트) 획득 → 레전드 등급 달성.
```

1. **몰입감**: 전래동화 캐릭터(도깨비, 구미호, 선녀 등)가 가이드 역할을 하는 스토리텔링
2. **자율성**: QR 스캔 한 번으로 셀프 진행 가능 (가이드 불필요)
3. **수집욕**: LP 포인트, 티어 달성, 코스 완료 배지
4. **다국어**: 한국어·일본어·영어 동시 지원

---

## 4. 전체 서비스 흐름

```
[구매 흐름]
랜딩페이지 → 코스 탐색 → 키트 상품 선택 → 결제(Toss/Stripe) → 배송 → QR코드 수령

[미션 흐름]
QR 스캔 → 언어 선택 → 스토리 인트로 → 현장 미션 수행 → 정답 입력/사진 업로드
→ LP 적립 → 다음 미션 → 코스 완료 → 배지 획득

[커뮤니티 흐름]
미션 완료 → 기록관(커뮤니티) 글 작성 → 좋아요/댓글 → LP 추가 적립

[보상 흐름]
LP 누적 → 티어 상승(씨앗→새싹→도깨비→선녀→신선) → 전설 상점에서 쿠폰/굿즈 교환
```

## 5. 기능 목록 (우선순위)

### Must (MVP)

| # | 기능 | 설명 |
|---|------|------|
| M1 | 코스 탐색 | 지역별·테마별 코스 목록, 상세 페이지 |
| M2 | 키트 구매 | Toss Payments(국내) + Stripe(해외) 결제 |
| M3 | QR 셀프 미션 엔진 | QR 스캔 → 미션 진행 → 정답 검증 → LP 적립 |
| M4 | LP·티어 시스템 | 포인트 누적, 5단계 티어, 배지 |
| M5 | 웹 커뮤니티 기록관 | 미션 완료 후 글쓰기, 사진 첨부, 댓글 |
| M6 | 제휴 연결 | 추천 숙소·교통·음식점 외부 링크 (affiliate_links) |
| M7 | 다국어 지원 | ko / ja / en (next-intl) |

### Should (2차)

| # | 기능 | 설명 |
|---|------|------|
| S1 | 전설 상점 | LP → 쿠폰/굿즈 교환 |
| S2 | B2B 주문 | 단체 키트 견적·발주 |
| S3 | 관리자 대시보드 | 주문, 미션 현황, 통계 |

### Could (3차 이후)

- 오프라인 이벤트 연계
- 여행사 파트너십 포털
- AR 미션 (카메라 인식)

## 7. 결제 구조

| 수단 | 대상 | 키 종류 |
|------|------|---------|
| Toss Payments | 국내 카드/계좌 | `NEXT_PUBLIC_TOSS_CLIENT_KEY`, `TOSS_SECRET_KEY` |
| Stripe | 해외 카드 | `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY`, `STRIPE_SECRET_KEY` |

- Stripe Webhook: `/api/webhooks/stripe`
- 주문 완료 후 `orders` 테이블에 저장, 키트 배송 처리

---

## 8. 다국어 전략

- 기본 locale: `ko`
- 지원: `ko`, `ja`, `en`
- 번역 파일: `/messages/{locale}.json`
- 모든 정적 데이터(`lib/data/*.ts`)의 텍스트 필드는 `{ ko, ja, en }` 객체로 작성
- URL 구조: `/{locale}/...` (예: `/ko/food/dupe/seoul`, `/ja/story`)

## 10. 비즈니스 모델

1. **키트 판매**: 코스별 물리 키트 (QR카드 + 소품 + 안내서) 판매
2. **B2B 납품**: 여행사·숙박업소 단체 주문
3. **제휴 수익**: 추천 숙소·교통 클릭당 수수료 (affiliate)
4. **전설 상점**: LP 교환용 굿즈 판매
